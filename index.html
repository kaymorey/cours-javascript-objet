Un objet c'est quoi ?

Un objet est une entité qui contient

  • Des propriétés qui contiennent des données
  • Des méthodes qui correspondent à des fonctions

Un objet c'est quoi ?

Une personne, par exemple, peut être caractérisée par

  • Des caractéristiques : Un nom, un prénom, une date de naissance...
  • Des fonctions : marcher, manger, parler...

JavaScript, un langage objet

  • Les strings, booléens, array sont des objets en javascript

            // name est un objet de type string
            const name = "John";

            // Accès à la propriété length de l'objet name
            name.length;

            // Appel de la méthode toUpperCase de l'objet name
            name.toUpperCase();

JavaScript, un langage objet

Il existe donc

  • les objets natifs existant dans le langage
  • les objets que vous allez créer

Créer des objets

1. Créer un objet littéral


            const person = {
              firstname: "John",
              lastname: "Doe",
              age: 30,
              speak: () => {
                console.log(`Hello, my name is ${this.firstname}`);
              }
            };

            // Accès à la propriété firstname de l'objet person
            person.firstname;

            // Accès à la méthode speak de l'objet person
            person.speak();

            // Ajout d'une propriété à l'objet person
            person.city = "Paris";

Créer des objets

  • Les propriétés d'un objet littéral sont caractérisées par un couple clé-valeur
  • On accède aux propriétés et aux méthodes de l'objet à l'aide du point "."
  • Un objet peut en contenir un ou plusieurs autres

            const person = {
              address: {
                street: "Rue de la Paix",
                city: "Paris"
              }
            };

            // Accès à la propriété city de l'objet address
            person.address.city;

Créer des objets

  • La limitation de l'objet littéral c'est qu'on va devoir en créer une copie pour obtenir un 2ème objet similaire
  • Si on doit créer plusieurs objets similaires on va donc plutôt faire en sorte que chaque objet hérite d'un modèle

Créer des objets

2. Utiliser un prototype


            function Person(firstname, lastname, age) {
              this.firstname = firstname;
              this.lastname = lastname;
              this.age = age;
            }

            Person.prototype.speak = function() {
              console.log(`Hello, my name is ${this.firstname}`);
            };

            const person1 = new Person("John", "Doe", 30);
            person1.speak(); // Affiche "Hello, my name is John"

            const person2 = new Person("Jane", "Doe", 25);
            person2.speak(); // Affiche "Hello, my name is Jane"

Créer des objets

3. Utiliser une classe


            class Person {
              constructor(firstname, lastname, age) {
                this.firstname = firstname;
                this.lastname = lastname;
                this.age = age;
              }

              speak() {
                console.log(`Hello, my name is ${this.firstname}`);
              }
            }

            const person1 = new Person("John", "Doe", 30);
            person1.speak(); // Affiche "Hello, my name is John"

            const person2 = new Person("Jane", "Doe", 25);
            person2.speak(); // Affiche "Hello, my name is Jane"

La notion d'héritage

  • De la même manière qu'en PHP, une classe peut hériter d'une autre en JavaScript
  • On utilise pour cela le mot-clé extends

La notion d'héritage


            class Person {
              constructor(firstname, lastname, age) {
                this.firstname = firstname;
                this.lastname = lastname;
                this.age = age;
              }

              speak() {
                console.log(`Hello, my name is ${this.firstname}`);
              }
            }

            class Student extends Person {
              constructor(firstname, lastname, age, school) {
                super(firstname, lastname, age);
                this.school = school;
              }

              study() {
                console.log(`I study at ${this.school}`);
              }
            }

            const student1 = new Student("John", "Doe", 30, "Harvard");
            student1.speak(); // Affiche "Hello, my name is John"
            student1.study(); // Affiche "I study at Harvard"

Le format JSON

  • Le JavaScript Object Notation (JSON) est un format de données
  • Il permet de structurer des données de façon semblable aux objets JavaScript
  • On accède donc aux données de la même manière qu'on le ferait pour un objet littéral

Le format JSON

Organiser votre code JavaScript à l'aide des objets


            import Modal from './Modal'
            import Menu from './Menu'

            class App {

                this.modal = new Modal()
